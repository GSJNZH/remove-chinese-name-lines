# 🧹 文本清理工具

一个简单轻量的网页工具，专门用于清理文本中**单独成行的人名**、**特定模式的行**（如“Options: 1”、“？？？”、“......”），并支持**按中文字符数量过滤**以及**行尾添加自定义文字**。所有处理均在浏览器本地完成，**不会上传任何数据**，保障隐私安全。

## ✨ 功能一览

### 1. 人名删除（核心）
- **整行删除**：删除完全由中文汉字组成且字数在指定范围内的行（默认 1~4 字，可调整）。
- **仅删行首人名**：只删除每行开头的人名部分，保留该行后面的对话或描述。

### 2. 额外整行删除规则（勾选即生效）
- **`Options: 数字`**：匹配如 `Options: 1` 的行（忽略大小写及前后空格）。
- **包含 `？？？`**：只要行内出现三个问号即删除整行。
- **只包含 `......`**：匹配仅由两个及以上英文句点组成的行。
- **中文字符数小于 5**：过滤掉中文字符少于 5 个的行（适用于去除短句、纯标点等）。
- **自定义正则**：可输入任意 JavaScript 正则表达式（需匹配整行），满足更复杂的删除需求。

### 3. 行尾添加文字
- 在清理完成后的每一行末尾（包括空行）添加自定义文字，方便批量添加标记、出处或编号。

### 4. 实时预览与复制
- 输入文本后点击“处理文本”，右侧实时显示结果。
- 一键复制处理后的文本，方便粘贴到其他地方。

## 📝 使用方法

1. **粘贴文本**：将需要清理的文本复制到左侧输入框中。
2. **调整参数**：
   - 选择人名删除模式（整行/行首）。
   - 设置人名汉字的字数范围（默认为 1~4）。
   - 勾选需要的额外删除规则。
   - 如需要，填写自定义正则并启用。
3. **行尾添加**（可选）：勾选“启用行尾添加”并输入要附加的文字。
4. **点击处理**：点击“处理文本”按钮，右侧结果区将实时显示清理后的内容。
5. **复制结果**：点击“复制结果”按钮，即可将结果粘贴到目标位置。

## ⚙️ 自定义与高级用法

### 自定义正则示例
| 目标 | 正则表达式 |
|------|------------|
| 删除空行 | `^\s*$` |
| 删除以 `[` 开头、`]` 结尾的行 | `^\[.*\]$` |
| 删除包含“测试”的行 | `测试` |

### 中文字符统计规则
- “中文字符数”统计的是 Unicode 范围内的汉字（`\u4e00` 到 `\u9fa5`），不包括标点、数字、英文字母。
- 例如：“你好！” 的中文字符数为 2；“Hello 世界” 的中文字符数为 1。

### 处理顺序
1. 人名删除（整行或行首）。
2. 额外规则删除（正则匹配）。
3. 中文字符数过滤（如果启用）。
4. 行尾添加文字（如果启用）。
