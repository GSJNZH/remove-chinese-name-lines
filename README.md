# 🧹 人名删除工具

一个简单轻量的网页工具，用于批量删除文本中**单独成行的人名**（中文），或仅删除行首的人名而保留后续内容。支持自定义汉字字数范围，所有处理均在浏览器本地完成，**不上传任何数据**。

## ✨ 功能

- **整行删除**：删除完全由汉字组成且字数在指定范围内的行（忽略首尾空格）。
- **仅删行首人名**：只删除每行开头的人名部分（保留该行后面的对话/描述）。
- 可自由设置最少/最多汉字数（默认 1~4 字）。
- 一键复制处理结果。

## 🚀 使用方法

1. 访问 GitHub Pages 地址：`https://<你的用户名>.github.io/remove-chinese-name-lines/`
2. 在左侧文本框粘贴需要清理的文本。
3. 选择删除模式，调整字数范围。
4. 点击「处理文本」，右侧实时显示结果。
5. 点击「复制结果」即可粘贴到其他地方。

## 📦 本地运行

直接下载 `index.html` 并用浏览器打开即可使用，无需任何依赖。

## ⚙️ 自定义正则

如果你想修改匹配规则（例如允许人名中包含点号或空格），可以自行编辑 `index.html` 中的 `buildRegex` 函数。核心正则如下：

```javascript
// 汉字范围
const charRange = '[\\u4e00-\\u9fa5]';
// 行首删除模式
new RegExp(`^\\s*${charRange}{min,max}`);
// 整行删除模式
new RegExp(`^\\s*${charRange}{min,max}\\s*$`);
